## Projet villages gaulois et romains

#### Description

Ce projet consiste à modéliser un univers de villages gaulois et romains en Java dans le cadre d'un projet Java pour un BUT de Seconde année.

L’objectif est d’implémenter une architecture orientée objet permettant de gérer :
Les habitants (Gaulois et Romains)
Les villages et leurs chefs
La potion magique et ses effets
Les interactions entre personnages (ex : combats, force augmentée, récolte de ressources, ...)

Le projet met en pratique la programmation orientée objet, la conception UML, l’architecture logicielle, et les tests unitaires.
